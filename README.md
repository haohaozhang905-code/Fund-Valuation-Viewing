# 实时基金估值与智能决策系统（纯前端）

一个面向个人投资者的纯前端基金看板：管理场外基金持仓、查看实时估值与收益，并结合AI给出盘中/收盘前决策建议。

> 设计原则：无登录、无后端、数据本地保存、开箱即用。

---

## 当前能力

### 基础持仓管理

- 新增 / 编辑 / 删除基金持仓（基金代码、成本价、持有份额）
- 基金列表与详情联动展示
- 本地持久化（`localStorage`）

### 实时估值与收益看板

- 顶部资产卡片：
  - 账户资产
  - 当日收益（金额 + 收益率）
  - 累计收益（金额 + 收益率）
- 持仓列表字段：
  - 当日收益
  - 当日估值 / 净值（带日期）
  - 持有收益
- 排序能力：按「当日收益」「持有收益」切换升降序

### AI 侧边栏（场外基金决策）

- 支持手动触发实时分析
- 支持自动触发（早盘结束、收盘前）
- 支持 OpenAI 兼容接口配置（模型、Endpoint、API Key）
- 支持 Markdown 渲染与重点加粗展示

### 交互与视觉

- 顶部标题栏固定
- AI 侧边栏可展开 / 收起
- 暗黑模式（记忆用户选择）
- 全局滚动条样式统一

## 收益计算口径

### 非交易时段

- 当日净值：取最新已公布净值
- 当日收益：`(最新净值 - 前一交易日净值) * 持有份额`
- 当日收益率：`(最新净值 - 前一交易日净值) / 前一交易日净值`

### 交易时段（09:30-11:30，13:00-15:00）

- 当日价格：优先估值，若不可用回退到净值
- 当日收益：`(当前估值(或净值) - 最新已公布净值) * 持有份额`
- 当日收益率：对应相对涨跌幅

### 累计收益

- 累计收益：`(当前价格 - 成本价) * 持有份额`
- 累计收益率：`(当前价格 - 成本价) / 成本价`
- 账户资产：`投资金额 + 累计收益`

## 本地存储键

- `fund_list`：持仓数据
- `fund_theme`：主题配置
- `ai_config_v1`：AI 接口配置
- `ai_auto_enabled`：AI 自动分析开关
- `ai_auto_trigger_cache_v1`：自动触发去重缓存
- `ai_bubble_pos_v1`：AI 悬浮球位置
- `index_snapshot_v1`：指数快照缓存

> 清空浏览器站点数据或 `localStorage` 会导致本地数据丢失。

## 已知限制

- 交易日判断仍基于工作日 + 时段，未接入权威节假日历
- 外部接口波动时，个别基金可能短时显示 `--`
- 当前为纯前端方案，不含多端同步与服务端校验
- AI 输出质量受模型与提示词影响，需用户二次判断

## 迭代规划（下一阶段）

以下是已确认的重点方向。

### 1) 实时基金估值预览图（折线图）

目标：在单基金详情与组合维度展示更直观的趋势变化。

计划内容：

- 单基金实时估值折线（分时/日内）
- 组合总收益曲线（当日与近几日）
- 支持暗黑主题下高对比配色
- 交互增强：悬停查看时间点数值、涨跌区间着色

验收标准（建议）：

- 图表不阻塞主线程，滚动与操作流畅
- 数据断点有合理补点或断线处理
- 同一基金切换展开时 1s 内出图（缓存命中）

### 2) 图片上传 + 自动解析入条目

目标：减少手动录入成本，支持从截图/拍照中提取基金信息并自动填充。

计划内容：

- 上传本地图片（拖拽 / 点击上传）
- OCR 提取关键字段（基金代码、名称、成本价、份额）
- 解析结果预览与人工确认后入库
- 对识别置信度低的字段给出高亮提示

验收标准（建议）：

- 常见券商/记账截图格式可稳定识别
- 提供“识别结果可编辑”流程，避免误入库
- 识别失败有可理解错误提示与重试引导

### 3) AI 总结更智能、更精准

目标：让 AI 输出从“通用描述”升级为“可执行策略”。

计划内容：

- 提示词分层：市场判断、组合暴露、操作建议、风险信号
- 引入上下文记忆：最近几次分析结论与执行结果
- 输出结构化模板（结论、理由、仓位建议、触发条件）
- 强化重点信息加粗与风险等级提示

验收标准（建议）：

- 建议可量化（仓位区间、触发条件、止损/止盈思路）
- 与当前持仓数据强相关，减少模板化空话
- 同等数据输入下输出风格稳定、可复盘
